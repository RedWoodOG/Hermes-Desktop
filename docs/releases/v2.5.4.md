# Hermes Desktop v2.5.4

**Release date:** 2026-04-28  
**Portable asset:** `HermesDesktop-portable-x64.zip` (from [GitHub Releases](https://github.com/RedWoodOG/Hermes-Desktop/releases))

## Highlights

### Soul system repair

- Treat `SOUL.md` as the runtime source of truth for the active identity label.
- Fixed Reset to Default to use the shipped `Default` soul template.
- Project `AGENTS.md` rules now flow into the assembled prompt context.
- Saved agent profiles now preserve and apply provider/model preferences when activated.
- AutoDream is registered and started so transcript consolidation can record mistakes, habits, and user-profile updates.

### Tests

- Added focused tests for first-run soul defaults, active soul naming, project rules in context, and profile provider/model activation.

## Upgrade notes

- Existing custom `SOUL.md` content is preserved and will show as **Custom** if it no longer matches a shipped soul template or active profile.
- After downloading the portable zip, switch between Souls and ask the same prompt to verify visible behavior differences.
